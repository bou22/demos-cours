# Exemple de backend

Voici deux exemple de backend pour montrer les autres possibiliter que php.

Il y en a un en [rust](https://www.rust-lang.org/) et un autre en [nodejs](https://nodejs.org/en/).

## Démarer un backend

Voici la commande pour démarer l'app avec un backend rust :
```bash
docker compose -f docker-compose.rust.yml up
```

Voici la commande pour démarer l'app avec un backend nodejs :
```bash
docker compose -f docker-compose.nodejs.yml up
```
